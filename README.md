# B & K Arquitectos

## Cómo correr el sitio localmente

Tienes dos opciones para correr el servidor local:

### Opción 1: Usando Python (Recomendado - No requiere instalación)

Python viene preinstalado en macOS. Simplemente ejecuta:

```bash
python3 -m http.server 8080
```

O usando el script npm:

```bash
npm start
```

Luego abre tu navegador en: `http://localhost:8080`

### Opción 2: Usando Node.js (si tienes Node instalado)

Si tienes Node.js instalado, puedes usar:

```bash
npm run serve
```

Esto abrirá automáticamente el navegador en `http://localhost:8080`

## Nota

Para detener el servidor, presiona `Ctrl + C` en la terminal.
